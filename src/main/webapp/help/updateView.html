Let Hudson/Jenkins Manage Workspace View: For most cases, leave this checked. Hudson/Jenkins will overwrite the workspace's view or stream to match the project's settings provided below.

If unchecked, you must manage the workspace's view or stream yourself. This is useful if, for this job, you need to checkout and build multiple projects from Perforce at once. (For those managing the workspace view themselves and looking for a starting point, the typical Hudson/Jenkins view is: '//depot/[PROJECTNAME]/... //[WORKSPACENAME]/...' You'll notice this is slightly different from typical workspaces. The [PROJECTNAME] is not normally included.)
